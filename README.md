# Gomoku

Gomoku is a project that implements the classic board game Gomoku. This project is written in Python and uses the `numpy` library.

## Installation

To install the project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/sboof911/Gomoku.git
    cd Gomoku
    ```

2. Install the required dependencies:
    ```sh
    pip install -e .
    ```


## Usage

To start the game, run the following command:
```
gomoku
```
